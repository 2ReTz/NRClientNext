Requirement
=====================================================================================
Python 3.8+
wxPython 4 (Phoenix) Unicode build

For more information about wxpython, check out https://www.wxpython.org/download.php

Quick setup
=====================================================================================
1. Create and activate a virtual environment (recommended).
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python PyNRClientX.py`

Installation
=====================================================================================

Directly extract the tarbar into a empty directory or overwrite the old
directory, that's ok.

Run
=====================================================================================

Execute:

python3 PyNRClientX.py

Notes
=====================================================================================
- Credentials entered in the Sign In dialog are saved to `connection.json` in this folder for convenience (plaintext). Delete it if you do not want local persistence.
